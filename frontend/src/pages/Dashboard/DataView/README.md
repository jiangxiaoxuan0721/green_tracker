# 数据展示界面实现

## 概述

本目录包含了数据展示界面的完整实现，使用户能够查看、过滤和分析他们自己的数据。所有数据请求都经过身份验证，确保用户只能访问自己的数据。

## 文件结构

- `DataView.jsx` - 主要的数据查看组件，包含数据表格、过滤器和分页功能
- `DataDetailModal.jsx` - 数据详情模态框，用于显示详细的数据信息
- `DataView.css` - DataView组件的样式
- `DataDetailModal.css` - 模态框组件的样式
- `README.md` - 本文档

## 功能特性

### 数据查看 (DataView.jsx)
- 用户身份验证和授权
- 数据过滤（按日期、设备、地块和数据类型）
- 分页显示数据
- 数据导出功能
- 删除数据功能
- 响应式设计，支持移动设备

### 数据详情 (DataDetailModal.jsx)
- 详细的数据信息显示
- 元数据查看
- 标签管理
- 多标签页界面

## API 集成

使用以下服务文件与后端API交互：
- `rawDataService` - 原始数据的CRUD操作
- `deviceService` - 设备管理
- `fieldService` - 地块管理

所有API请求都包含用户ID，确保用户只能访问自己的数据。

## 安全性

- 所有数据请求都经过身份验证
- 用户ID包含在每个API请求中
- 后端验证用户权限
- 前端使用认证上下文管理用户状态

## 使用方法

1. 用户登录后，导航到"数据查看"页面
2. 使用过滤器筛选数据
3. 点击"详情"按钮查看完整数据信息
4. 使用分页控件浏览大量数据
5. 使用"导出数据"按钮下载数据

## 自定义选项

可以通过修改以下内容来自定义界面：
- `DataView.css` - 调整表格和过滤器样式
- `DataDetailModal.css` - 调整模态框样式
- 过滤器选项在`DataView.jsx`中定义
- 表格列可以在表格定义中添加或删除
